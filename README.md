# OpenFIIs

Dashboard web para simulacao e acompanhamento de carteiras de FIIs.

## Stack

- Next.js + TypeScript
- Recharts para graficos
- Framer Motion para animacoes
- Lucide React para icones
- Supabase preparado via variaveis de ambiente
- Deploy pronto para Vercel

## Desenvolvimento

```bash
npm install
npm run dev
```

Crie um `.env.local` baseado em `.env.example` quando for conectar ao Supabase.
